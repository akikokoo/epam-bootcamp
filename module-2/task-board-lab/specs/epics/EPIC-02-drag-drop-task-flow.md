# Epic: Drag-and-Drop Task Movement

<!-- Keep this concise and outcome-focused. Example: "Self-Service Team Onboarding" -->

## 1. Epic Title

Drag-and-Drop Task Movement

## 2. Description (2-3 sentences)

Enable users to move tasks between columns and reorder tasks within a column via drag-and-drop. This epic delivers the fast, tactile interaction model required for daily task flow and supports the performance success metric for task operations. It builds on the core board and task list UI.

## 3. Primary Persona (who benefits most)

Emre Kaya (Indie Product Builder)

## 4. Success Criteria (measurable outcomes)

1. Drag-and-drop move or reorder completes in under 150 ms for a 200-task board.
2. 95% of drag actions result in correct task position and column assignment in usability tests.
3. Drag interactions maintain smooth UI performance (>60 FPS) during task moves.

## 5. Scope/Complexity (S/M/L estimate)

**Estimate:** M

## 6. Dependencies (what must exist first)

- Core Kanban board layout and task list rendering.
- Task ordering model per column.
- Selection or active task state handling.
