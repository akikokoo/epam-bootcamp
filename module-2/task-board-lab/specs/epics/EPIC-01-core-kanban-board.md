# Epic: Core Kanban Board & Task CRUD

<!-- Keep this concise and outcome-focused. Example: "Self-Service Team Onboarding" -->

## 1. Epic Title

Core Kanban Board & Task CRUD

## 2. Description (2-3 sentences)

Deliver the foundational three-column board (To Do, In Progress, Done) and task creation, editing, and deletion flows so a solo developer can capture and organize work quickly. This epic establishes the task data model, board layout, and basic interactions needed to start using the app immediately. It directly supports the “create first task in under 2 minutes” goal.

## 3. Primary Persona (who benefits most)

Aylin Demir (Solo Full-Stack Developer)

## 4. Success Criteria (measurable outcomes)

1. 90% of test users can create their first task within 2 minutes of first launch.
2. Creating, editing, or deleting a task completes in under 300 ms for a 200-task board.
3. Board renders the three columns with up to 200 tasks in under 1 second on a mid-range laptop.

## 5. Scope/Complexity (S/M/L estimate)

**Estimate:** M

## 6. Dependencies (what must exist first)

- React + Vite + TypeScript project scaffold.
- Task data model (id, title, optional description, status, order).
- Base UI components for columns and task cards.
